Sobre o Projeto

O EduControl é um sistema web desenvolvido para auxiliar escolas no gerenciamento de notas, alunos, professores e turmas. O sistema permite o lançamento de notas e o cálculo automático da média final.

Funcionalidades

  • ✔ Sistema de Login com níveis de acesso
  • ✔ Cadastro de alunos
  • ✔ Cadastro de professores
  • ✔ Cadastro de turmas e matérias
  • ✔ Lançamento de notas
  • ✔ Cálculo automático da média
  • ✔ Relatório de aprovação ou reprovação

Tipos de Usuários

  • Administrador: Gerencia todo o sistema.
  • Professor: Lança notas e consulta turmas.
  • Aluno: Visualiza suas notas.

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript (para melhorias futuras)
  • Banco de Dados MySQL (planejado)
